# הגדת פסח מודרנית — Modern Passover Haggadah 2026

A Hebrew-language, interactive web-based Passover Haggadah designed for a modern Israeli Seder. Built with plain HTML, Tailwind CSS, and custom styles — no build tools required.

## Features

- Right-to-left (RTL) Hebrew layout
- 8-page seder flow with prev/next navigation
- Wine-cup prompts and song links (Spotify)
- Quotes from Israeli poets, authors, and thinkers
- Responsive design (mobile & desktop)
- Custom fonts: *Frank Ruhl Libre* (headings) & *Assistant* (body)

## Getting Started

No installation needed. Open `index.html` in any modern browser — it redirects automatically to `page1.html`.
